# Documentación del uso de IA

## Herramienta utilizada

ChatGPT

Durante el desarrollo del proyecto usé ChatGPT para resolver errores, entender partes del código y mejorar el diseño visual de la página.



## Ejemplos de prompts utilizados
1. Corrección de rutas CSS y estructura del proyecto

Prompt:

“Mirá el link y tengo guardado mi css acá. ¿Está bien?”

Resultado obtenido:

Corrección de rutas relativas.
Organización correcta de carpetas para GitHub Pages.
Mejora de estructura del proyecto.

2. Ayuda con Git y GitHub

Prompt:

“No entiendo qué pasa con git push”

Resultado obtenido:

Configuración de upstream.
Resolución de conflictos con GitHub.
Uso correcto de git pull, git commit y git push.

3. Explicación de JavaScript paso a paso

Prompt:

“Necesito que me expliques qué hace cada cosa”

Resultado obtenido:

Explicación detallada de:
funciones
loops for
arrays
generación de colores HEX
manipulación del DOM
eventos en JavaScript

4. Mejora del diseño visual

Prompt:

“Quiero fondo oscuro, letras normales y diseño minimalista”

Resultado obtenido:

Implementación de estilo minimalista oscuro.
Mejora de botones y selects.
Ajustes tipográficos y espaciados.

## Resultado final

La IA me ayudó principalmente a:

entender mejor JavaScript,
resolver errores técnicos,
organizar el proyecto,
y mejorar la interfaz visual.
