# Proyecto M1 - Generador de Paletas

Aplicación web interactiva que genera paletas de colores aleatorias para propuestas visuales y branding.

## Funcionalidades

- Generación aleatoria de colores
- Selección de tamaño de paleta
- Copia de códigos HEX
- Bloqueo de colores
- Feedback visual (toast)
- Diseño dark minimalista

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

## Demo del proyecto

https://florencianarvaez.github.io/ProyectoM1_FlorenciaNarvaez/

## Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Abrir el archivo index.html en el navegador

## Decisiones técnicas

- Se utilizó manipulación del DOM para generar dinámicamente las tarjetas de colores.
- Los colores bloqueados se almacenan en un array.
- Se implementó feedback visual mediante un toast.
- Se utilizó un diseño dark minimalista para mejorar la experiencia visual.
